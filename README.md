# WatchCrab

看门蟹，一个监控各平台粉丝数量的桌面程序。

## 安装
```
yarn install
```

### 启动
```
yarn dev
```

### 编译
```
yarn build
yarn build:win32
yarn build:win64
yarn build:mac
...
```
